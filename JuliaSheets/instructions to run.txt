1. Install python (3.10). Make sure it is in the PATH.

2. Install jupyter-notebook through pip:
        pip install notebook

3. Install Julia (1.9.0). Make sure it is in the PATH.

4.  a. Open Julia in command prompt or in powershell
    b. Type "]" to enter package mode
    c. Type and run "add Interact IJulia PyPlot LaTeXStrings WebIO"

5.  a. Install WebIO extension through pip:
        python3 -m pip install --upgrade webio_jupyter_extension
    b. Ensure that WebIO extension is installed for jupyter notebook:
        jupyter nbextension list
        should return an entry
         "webio-jupyter-nbextension/nbextension enabled
      - Validating: ok"

6. Launch jupyter notebook from command prompt/powershell:
        jupyter notebook

7. Navigate to directory with Julia Jupyter notebooks and open them.
